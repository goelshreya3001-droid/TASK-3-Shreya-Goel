## 📌 Project Overview

**StudySync** is a responsive and interactive educational landing page built using **HTML, CSS, and JavaScript**. The project focuses on delivering a clean user interface, responsive layouts, and engaging user interactions through DOM manipulation and event handling.

The website adapts seamlessly across desktops, tablets, and mobile devices while providing interactive features that enhance the overall user experience.

This project was developed to demonstrate:

* 📱 Responsive Web Design

* ⚡ JavaScript Interactivity
